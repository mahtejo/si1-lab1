O que é SI 1?

A disciplina Sistemas de Informação 1 é uma disciplina que vai te ajudar a dar os primeiros passos a fim de se familiarizar com o ambiente de desenvovimento Web, através de discussões técnicas sobre projetos de software. A disciplina possui um foco maior em cima de tecnicas utilizadas para melhorar o reuso e a flexibilidade de um software produzido utilizando Orientação ao Objeto. Isso permite que o aluno saia da disciplina sabendo desenvolver sistemas web.



Porque cursar SI1?

  1. O conteúdo que a disciplina trata é indispensável nos dias de hoje;
  2. O professor é bem bacana;
  3. A metodologia é excelente para um bom aprendizado;
  4. Você precisa passar por essa disciplina se quiser se graduar em Ciência da Computação na UFCG.
