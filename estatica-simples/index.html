O que é S.I. 1?

A disciplina Sistemas de Informação 1 é uma disciplina que vai te ajudar a dar os primeiros passos a fim de se familiarizar com o ambiente de desenvovimento Web, através de discussões técnicas sobre projetos de software. A disciplina possui um foco maior em cima de tecnicas utilizadas para melhorar o reuso e a flexibilidade de um software produzido utilizando Orientação ao Objeto Fundamentos; princípios de design (modularidade, coesão, acoplamento); paradigmas de design (funcional, orientado a objetos, outros); projeto orientado a objetos; linguagem de modelagem; abstração, herança, interfaces; bibliotecas e frameworks; padrões de design; noções de arquitetura de software.
